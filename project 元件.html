電子零件自動化辨識系統

基於 iPhone 17 Pro 與 YOLOv8 之實作研究報告

1. 檢測物件

本專題針對 PCB 電路板上的核心元件進行辨識，確保生產線上的元件放置正確且無瑕疵：

  • 電阻 (Resistor)：檢測表面阻值標記及焊點狀態。
  • 電容 (Capacitor)：辨識陶瓷與電解電容的位置及極性。
  • 電晶體 (Transistor)：確認接腳是否正確對齊。
  • IC 晶片：掃描封裝表面標籤與接腳完整性。

2. 檢測方法

📸 影像獲取：iPhone 17 Pro

採用 iPhone 17 Pro 搭載的 48MP 主鏡頭。利用其先進的宏觀攝影模式與光學防手震技術，在 2cm 近距離下擷取高解析度、低噪訊的原始零件影像，為模型提供優質的訓練素材。

🧠 辨識核心：YOLOv8

核心算法採用 YOLOv8 Nano 版本。該模型在兼顧推論速度的同時，具備優異的物件特徵提取能力，能針對 PCB 上密集的電子零件實現毫秒級的即時偵測。

3. 檢測流程

1
資料擷取： 使用 iPhone 17 拍攝 PCB 影像並同步至伺服器。
2
影像降噪： 執行中值濾波器以去除環境雜訊。
3
AI 推論： 透過 YOLOv8 進行目標定位與類別分類。
4
判定結果： 根據信心度與幾何特徵進行良品(OK)或不良品(NG)判定。

4. 影像處理

中值濾波器 (Median Filter) 去噪實作

針對 iPhone 17 在工業環境光源下產生的「椒鹽雜訊」(Salt-and-Pepper Noise)，我們應用了 中值濾波器。其原理是將鄰域像素排列並取中值，能有效保護邊緣不被模糊化的同時移除突發性亮點。 

# 基於 OpenCV 的前處理程式碼
def preprocess_image(frame):
    # 應用 5x5 中值濾波器去除噪訊
    denoised = cv2.medianBlur(frame, 5) 
    
    # 色彩空間轉換與增強
    lab = cv2.cvtColor(denoised, cv2.COLOR_BGR2LAB)
    l, a, b = cv2.split(lab)
    
    # CLAHE 對比度受限自適應直方圖均衡化
    clahe = cv2.createCLAHE(clipLimit=3.0, tileGridSize=(8,8))
    cl = clahe.apply(l)
    
    return cv2.cvtColor(cv2.merge((cl, a, b)), cv2.COLOR_LAB2BGR)
